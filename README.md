## 🚀 Bắt Đầu Nhanh

### Cách mở các bài tập
1. Clone repository này:
```bash
git clone <repository-url>
```

2. Mở file bất kỳ trong trình duyệt:
   - Chuột phải vào file `.html` → "Open with Live Server" (nếu có extension)
   - Hoặc kéo thả file vào trình duyệt

### Cách làm bài
- Mở file HTML trong code editor (VS Code, Sublime Text, etc.)
- Sửa code theo hướng dẫn của bài
- Xem kết quả trong trình duyệt bằng cách Refresh (F5 hoặc Ctrl+R)

## 💡 Kiến Thức Cần Biết

### HTML - Hyper Text Markup Language
HTML là ngôn ngữ để tạo cấu trúc và nội dung của trang web.
- Dùng thẻ để đánh dấu nội dung
- Công dụng: định nghĩa heading, paragraph, link, hình ảnh, form, v.v.

### CSS - Cascading Style Sheets
CSS dùng để trang trí và bố cục cho HTML.
- Điều chỉnh màu sắc, font, kích thước, vị trí
- Tạo responsive design cho nhiều thiết bị
